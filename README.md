# Weatherly App 🌤️

**Weatherly** is a premium, modern, and sleek weather tracking application built with Flutter. It provides users with real-time weather updates, accurate forecasts, and a delightful user experience with its glassmorphism-inspired design.

## 🚀 Key Features
- **Real-time Weather:** Get live temperature and weather updates for any city.
- **Sleek UI/UX:** A premium aesthetic with vibrant gradients and glassmorphism effects.
- **Global Search:** Easily search for weather information of any city in the world.
- **Responsive Design:** Optimized for all screen sizes and devices.

## 🛠 Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **API:** OpenWeatherMap API
- **State Management:** Provider / Riverpod
